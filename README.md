# Java Backend Projects

Collection of Java backend development projects focused on object-oriented programming, CRUD systems, REST APIs, and database integration.

## Projects

### 1. Student Database System
A console-based Java application that manages student records using object-oriented programming principles and CRUD operations.

#### Features
- Add students
- View students
- Search students by ID
- Remove students
- Console menu system

#### Technologies Used
- Java
- IntelliJ IDEA
- ArrayLists
- OOP Principles

#### Concepts Practiced
- Classes and objects
- Encapsulation
- Constructors
- Methods
- Lists and collections
- User input handling

#### Future Improvements
- Database integration with PostgreSQL/MySQL
- Spring Boot REST API
- File persistence
- Docker support
- Authentication and validation
