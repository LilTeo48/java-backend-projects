# Projects

## 1. Student Database System

A console-based Java application that manages student records using object-oriented programming principles and CRUD operations.

### Features

- Add new student records
- View all students
- Search students by ID
- Search students by name
- Update existing student records
- Remove students by ID
- Prevent duplicate student IDs
- Validate GPA input between 0.0 and 4.0
- Display total student count
- Sort students by GPA from highest to lowest
- Clean formatted console output
- Menu-driven command-line interface
